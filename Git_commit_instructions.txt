instructions for committing a change in GitHub

1. Modify the file/create new file
2. stage the file using [git add]
3. commit the file and add a message using [git commit -m] or just [git commit]
4. push the file using [git push]

make sure to check status on file using [git status] as you go along 


You can create a new branch using the command [git checkout -b new_branch_name]
